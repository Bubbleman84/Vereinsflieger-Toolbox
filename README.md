# Vereinsflieger-Toolbox

Hallo zusammen,

ich möchte hier ein paar Basisklassen für den Zugriff auf Vereinsflieger.de über die REST-Api zur allgemeinen Verwendung zur Verfügung stellen. 
Ich finde das nicht jeder dieses Klassen selbst erfinden muss und man sich so viel Arbeit sparen kann.

Was natürlich benötigt wird sind sowohl Zugangsdaten für Vereinsflieger.de als auch ein Appkey, welche man direkt dort im Support beantragen kann.

Ich freue mich über eure Unterstützung dabei das Projekt immer weiter und weiter aufzubauen, so das bald ja vielleicht alle API-Möglichkeiten von Vereinsflieger über die Klasse abgerufen werden können.
